Beehive DLNA 1.0.1

DLNA клиент для просмотра видео.

Версия 1.0.1

Описание

Бесплатное Android (4.1+) приложение для обзора домашней коллекции фильмов и сериалов, хранящихся на компьютере, и последующем просмотре на видеоплеере. Из ключевых особенностей — возможность воспроизводить все файлы в директории, используя сторонний плеер. Для работы приложения необходим развернутый в локальной сети DLNA сервер и любой android-видеоплеер, поддерживающий плейлисты m3u (например VLC for Android) {Скачать APK}

Скриншоты

DLNA?

DLNA (англ. Digital Living Network Alliance) — набор стандартов, позволяющих совместимым устройствам передавать и принимать по домашней сети различный медиа-контент (изображения, музыку, видео). {Википедия}

Выбор DLNA сервера

Plex Media Server — кроссплатформенный сервер, поддерживающий транскодирование. Умеет брать метаданные из сети. Прост в установке и настройке, при этом достаточно мощный. {Страница загрузки}
Kodi — самый популярный и полностью бесплатный комбайн, который также умеет раздавать видео контент через DLNA. Запускается как замена окружения рабочего стола, нет фонового режима. {Страница загрузки}
Serviio — DLNA-stream сервер для Windows, Linux, MacOs. Транскодирование и веб-клиент доступны только в Pro версии. {Страница загрузки}
Emby Server — медиа сервер, созданный для организации медиаконтента и его раздачи на другие устрйоства {Страница загрузки}
